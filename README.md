# StoryStack

StoryStack is a full-stack blogging platform that enables users to create, publish, edit, and manage blog posts through a clean and intuitive interface. The application features secure user authentication, image uploads, and a responsive design.

## 🚀 Features

- User Authentication & Authorization
- Create, Read, Update, and Delete Blogs
- Image Upload Support
- Cookie-Based Authentication
- Responsive User Interface
- Server-Side Rendering with EJS
- MongoDB Database Integration
- MVC Architecture

## 🛠️ Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

### Frontend
- EJS
- Bootstrap
- HTML
- CSS
- JavaScript

### Authentication
- JWT (JSON Web Token)
- Cookies
